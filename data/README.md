Contains the raw dataset used for the analysis.

- FemaleEmploy.xlsx — Annual US household data (1913–1984) from the FERTIL3 
  dataset. Includes chld (dependent children per 1,000 women), exem (real tax 
  exemptions), wartm (WWII dummy), eduprog (education programme dummy), 
  and t (time trend). Sheet used: FERTIL3.
