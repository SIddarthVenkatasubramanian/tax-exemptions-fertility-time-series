# Tax Exemptions & Fertility — Time Series Analysis of US Households (1913–1984)

## Overview
This study examines whether real federal tax exemptions for dependent children 
influenced fertility decisions in the United States from 1913 to 1984. Using annual 
time-series data on 72 observations, the analysis estimates the short-run and 
long-run effects of tax policy on birth rates, controlling for World War II and the 
introduction of women's educational programmes.

## Research Question
Do firm-specific fiscal incentives — specifically real tax exemptions for dependents — 
have a statistically significant effect on fertility behaviour over time?

## Variables
| Variable | Description |
|---|---|
| `chld` | Dependent children aged 0–3 per 1,000 women (dependent variable) |
| `exem` | Real dollar value of tax exemptions (key explanatory variable) |
| `wartm` | Dummy variable: World War II period (1941–1945) |
| `eduprog` | Dummy variable: Women's educational programme introduced in 1963 |
| `t` | Time trend (with t², t³ for non-linear effects) |

## Methodology
- Augmented Dickey-Fuller (ADF) stationarity testing
- Trend model comparison (Linear, Quadratic, Cubic)
- Static OLS Regression
- Finite Distributed Lag (FDL) Model with 4 lags of exem
- Long-Run Propensity (LRP) estimation using delta method
- Changes-Based Model (first differences) as robustness check
- Diagnostics: Durbin-Watson, Breusch-Godfrey, Breusch-Pagan tests

## Key Findings
- Both chld and exem are non-stationary in levels; exem is I(1), chld is 
  trend-stationary
- Static model: tax exemptions have a significant positive short-run effect 
  (coef = 0.142, p = 0.001); a $12 increase raises fertility by ~1.70 per 1,000 women
- FDL model: improves fit (R² = 0.871); lagged terms jointly significant 
  (F-test p = 0.004)
- Long-Run Propensity (LRP = 0.186, p < 0.001): sustained exemptions have a 
  meaningful positive cumulative effect on fertility
- Changes-based model: negative short-run effect (−0.095, p = 0.011), consistent 
  with lagged adjustment rather than contradiction
- WWII (−15.4) and women's education programme (−29.3) both exert strong 
  negative effects on fertility
- Levels models suffer from autocorrelation and heteroskedasticity; 
  changes-based model satisfies CLM assumptions
